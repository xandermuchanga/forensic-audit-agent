# Demo Scenario

## Scenario
Q1 2025 audit of vendor payments exceeding $10,000.

## User Prompt
"Review Q1 2025 vendor payments above $10,000 for fraud risk."

## System Flow
1. Intent interpreted by the main agent
2. Read-only query generated and validated
3. Data quality checks performed
4. Duplicate payments and vendor anomalies detected
5. Fraud patterns analyzed
6. Executive forensic audit report generated

## Output
- Risk classification (Low / Medium / High)
- Evidence-backed findings
- Actionable recommendations for finance and audit teams
